# 自动按键器 (Auto Clicker)

一个用 C 语言编写的 Windows 自动按键+鼠标连点工具，支持自定义键盘按键序列、鼠标连点速度/类型、独立全局热键开关、循环执行，并通过配置文件进行设置。无需复杂 GUI，编辑文本文件即可完成所有配置。

## 功能特点

- ✅ **键盘自动按键**：自定义按键序列（支持单个按键、组合键如 Ctrl+C、Alt+Tab），可单独设置每个按键的延时，支持循环执行。
- ✅ **鼠标连点**：独立控制，支持左键/右键/中键点击，速度可调（极速10ms、高效100ms、普通500ms、自定义毫秒数）。
- ✅ **独立全局热键**：键盘默认 `F6`，鼠标默认 `F4`，两者互不干扰，均可在配置文件中修改。
- ✅ **配置文件驱动**：所有设置（按键序列、热键、鼠标参数）均通过 `config.txt` 编辑，无需对话框，永不卡死。
- ✅ **支持特殊按键**：Delete、Insert、Home、End、PageUp、PageDown、PrintScreen、Pause、NumLock、ScrollLock。
- ✅ **支持主键盘数字键（0-9）和小键盘数字键（Num0-Num9）**。
- ✅ **使用扫描码（Scan Code）发送按键**，兼容大多数游戏。
- ✅ **静态链接**，单个 exe 文件，无依赖，支持 Windows XP 及以上。

## 首次使用

1. 运行 `AutoClicker.exe`（建议以管理员身份运行，否则部分游戏可能无法接收模拟按键）。
2. 程序会在同目录下自动生成 `config.txt` 配置文件。
3. 用记事本打开 `config.txt`，按你的需求修改配置。
4. 点击程序窗口上的「重新加载」按钮使设置生效。
5. 按对应的热键开始/停止（键盘默认 F6，鼠标默认 F4）。

## 配置文件详细说明

### 键盘配置

| 配置项 | 说明 |
|--------|------|
| `Hotkey=` | 键盘全局开关键，支持 F1-F12、A-Z、0-9、Enter、Tab、Space、Esc、Backspace、Ctrl、Alt、Shift、Win 等 |

每行一个按键，空格后跟延时（毫秒），如 `A 500`

**示例：**
```ini
Hotkey=F6

A 500
Ctrl+C 200
Delete 400
Num1 300
```

### 鼠标配置

| 配置项 | 说明 |
|--------|------|
| `MouseHotkey=` | 鼠标连点全局开关键 |
| `MouseClickType=` | 点击类型：left（左键）、right（右键）、middle（中键） |
| `MouseDelay=` | 连点间隔（毫秒），如 100 |

**示例：**
```ini
MouseHotkey=F4
MouseClickType=left
MouseDelay=100
```

### 完整配置文件示例

```ini
# ========== 自动按键器配置 ==========
# 键盘开关键
Hotkey=F6

# 键盘按键序列
A 500
Ctrl+C 200
Delete 400

# 鼠标连点配置
MouseHotkey=F4
MouseClickType=left
MouseDelay=100
# ===================================
```

## 使用方法

### 图形界面操作

- **编辑配置**：点击按钮会用记事本打开 `config.txt`，修改后保存。
- **重新加载**：使修改后的配置文件立即生效。
- **开始键盘/停止键盘**：手动控制键盘自动按键。
- **循环执行**：勾选后键盘序列将无限循环。
- **鼠标区域**：可选择点击类型、速度模式（极速/高效/普通/自定义），手动输入毫秒数，通过按钮或热键控制开始/停止。
- **热键显示**：窗口底部会显示当前键盘和鼠标的热键。

### 热键控制

- **键盘默认 F6**：按下开始键盘自动按键，再按一次停止。
- **鼠标默认 F4**：按下开始鼠标连点，再按一次停止。

### 修改热键

编辑 `config.txt` 中的 `Hotkey=` 或 `MouseHotkey=`，保存后点击「重新加载」即可。

## 常见问题

**Q: 为什么在游戏中没有反应？**  
A: 请以管理员身份运行 AutoClicker.exe（右键 -> 以管理员身份运行）。部分游戏还需要关闭杀毒软件或使用硬件模式。

**Q: 键盘和鼠标可以同时运行吗？**  
A: 可以，两者独立控制，互不干扰。

**Q: 程序会报毒吗？**  
A: 由于程序模拟按键，部分杀毒软件可能会误报。这是正常现象，代码完全开源，你可以自行审查或编译。

**Q: 如何让程序开机自启？**  
A: 将 AutoClicker.exe 的快捷方式放入 启动 文件夹（shell:startup）即可。

**Q: 配置文件中的注释可以删除吗？**  
A: 可以，`#` 开头的行都是注释，删除不影响功能。

**Q: 鼠标速度自定义输入后，下次启动会记住吗？**  
A: 当前版本中，界面上的速度修改不会自动写入配置文件。如需永久保存，请直接编辑 `config.txt` 中的 `MouseDelay=` 值。
